Para rodar o projeto localmente, siga estas instruções:

1.  **Descompacte o arquivo:** Extraia o conteúdo do arquivo `sonora_app_modificado.tar.gz` para uma pasta de sua escolha.
2.  **Navegue até o diretório do projeto:** Abra um terminal ou prompt de comando e use o comando `cd` para entrar na pasta onde você extraiu os arquivos. Por exemplo, se você extraiu para `C:\sonora-app`, digite `cd C:\sonora-app`.
3.  **Instale as dependências:** Se você tiver o `pnpm` instalado, execute `pnpm install`. Caso contrário, você pode usar `npm install` ou `yarn install` (certifique-se de ter o Node.js e o npm/yarn instalados).
4.  **Configure as variáveis de ambiente:**
    *   Crie um arquivo chamado `.env` na raiz do projeto (na mesma pasta que `package.json`).
    *   Adicione as seguintes linhas ao arquivo `.env`:
        ```
        VITE_SPOTIFY_CLIENT_ID=seu_client_id
        VITE_SPOTIFY_CLIENT_SECRET=seu_client_secret
        ```
    *   Substitua `seu_client_id` e `seu_client_secret` pelas suas credenciais da API do Spotify. Você pode obtê-las registrando um aplicativo no [Spotify Developer Dashboard](https://developer.spotify.com/dashboard/applications).
5.  **Inicie o servidor de desenvolvimento:** Execute o comando `pnpm run dev` (ou `npm run dev` / `yarn dev` se você usou npm/yarn para instalar as dependências).
6.  **Acesse o aplicativo:** Abra seu navegador e vá para `http://localhost:5173` (ou a porta que o Vite indicar no terminal).

Se encontrar algum problema, verifique se você tem o Node.js e o pnpm (ou npm/yarn) instalados corretamente e se as variáveis de ambiente estão configuradas como descrito acima. Se os problemas persistirem, por favor, me informe os detalhes para que eu possa ajudar.
